A highly animated, glassmorphism-themed conversational AI interface powered by the Google Gemini 1.5 Flash model. Designed as a lightweight, single-file frontend application with secure client-side API key management.

## 🌟 Features

* **Advanced AI Conversation:** Real-time, context-aware responses driven by the Gemini API.
* **Modern UI/UX:** Dark-mode glassmorphism aesthetics with dynamic background animations and glowing accent UI elements.
* **Client-Side Security:** "Vault" approach for API key initialization, storing credentials securely in the browser's local storage rather than hardcoding them in the repository.
* **Fully Responsive:** Fluid layout that adapts seamlessly to desktop, tablet, and mobile environments.
* **Zero Backend Required:** Runs entirely in the browser using native JavaScript `fetch` architecture.

* **Icons & Fonts:** FontAwesome 6.4.0, Google Fonts (Outfit)
* **Intelligence:** Google Generative AI (Gemini 1.5 Flash API)
